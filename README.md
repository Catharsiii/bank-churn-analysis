# Bank Customer Churn Analysis

## Описание
Продуктовый аналитический проект по анализу оттока клиентов банка.
Цель - помочь продуктовой команде банка выявить ключевые факторы оттока и разработать рекомендации по его снижению.

## Датасет
- Источник: Kaggle - Bank Customer Churn Prediction
- 10 000 клиентов, 12 признаков
- Целевая переменная: `churn` (1 — ушёл, 0 — остался)
- Дисбаланс классов: 80% / 20%

## Стек
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- PostgreSQL / DBeaver
- Power BI

## Структура репозитория
- `notebooks/` — EDA в Jupyter Notebook
- `sql/` — SQL-запросы в PostgreSQL
- `dashboard/` — интерактивный дашборд Power BI
- `data/` — датасет

## Ключевые инсайты
- Общий отток: 20,37%
- Германия — аномально высокий отток: 32,44% (vs 16% в других странах)
- Женщины уходят чаще: 25% vs 16% у мужчин
- Неактивные клиенты уходят вдвое чаще: 41% vs 24%
- Возраст — главный предиктор оттока (корреляция 0.29)

## Бизнес-рекомендации
1. Приоритет — работа с неактивными клиентами: программы реактивации
2. Отдельная стратегия удержания для немецкого рынка
3. Персонализированные офферы для клиентов 40-60 лет
